Klucz API Anthropic

Wklej klucz API uzyskany na console.anthropic.com → API Keys. Klucz jest zapisywany lokalnie w Twojej przeglądarce.

Klucz nie jest wysyłany nigdzie poza API Anthropic.

Jak uzyskać klucz API

1Wejdź na console.anthropic.com i zaloguj się lub zarejestruj.
2W menu po lewej kliknij API Keys.
3Kliknij Create Key i nadaj mu nazwę np. "PERSONALNI raport".
4Skopiuj klucz i wklej go powyżej.

Instrukcja

Jakie pliki wgrywać z Booksy

1Lista wizyt — najważniejszy plik. Każda wizyta z datą, klientem, trenerem, usługą, wartością i statusem. Booksy Biz: Statystyki → Wizyty → Pobierz.
2Lista klientów — dane klientów: liczba wizyt, daty aktywności, wartość, grupy. Booksy Biz: Klienci → Pobierz.
3Efektywność pracowników — wizyty i % powracających klientów per trener.
4Czas pracy pracowników — grafik i czas wizyt per trener (do % wypełnienia grafiku).
5Raport wizyt — zbiorcze podsumowanie obłożenia i dni/godzin z najlepszymi wynikami.

Agent rozpoznaje typ pliku automatycznie po zawartości — nazwa pliku nie ma znaczenia.

Typy raportów

🏢Cała firma — kompleksowy raport studia: wizyty, obłożenie, klienci, retencja, LTV, rekomendacje.
👤Pracownik — wybierz konkretnego trenera lub zostaw puste dla raportu wszystkich. Każdy dostaje osobną sekcję z wynikami i rekomendacjami.
🎯Usługa — wybierz konkretną usługę lub zostaw puste dla analizy wszystkich usług.
